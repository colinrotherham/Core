Core
====

### What is it?
Try the demo site built using Core: https://apps.colinr.com/core/

### Initial setup

Install Node.JS

As root/administrator:

```
npm install -g gulp-cli
```

Add Core to your project:
```
npm install @colinrotherham/core
```

### Building, Testing
See a sample project built using Core:
